定义串接口 IString 两版本相同

串的抽象数据类型，定义所有操作方法签名

▼
📋 实训目标：理解串的 ADT（抽象数据类型），接口定义了 length、concat、compareTo、substring、indexOf、charAt、insert、delete 共 8 个核心操作。
src/string/IString.java 
/**
 * 串的接口定义（ADT）
 * 顺序串和链串都实现此接口，保证接口一致性
 */
public interface IString {

    /** 返回串的长度（字符个数） */
    int length();

    /** 将串 t 联接到当前串末尾，返回新串 */
    IString concat(IString t);

    /** 按字典序比较两串大小，返回差值（负=小，0=相等，正=大） */
    int compareTo(IString t);

    /** 取子串 [begin, end)，返回新串 */
    IString substring(int begin, int end);

    /** 从 begin 位置开始查找子串 t，返回首次出现位置，未找到返回 -1 */
    int indexOf(IString t, int begin);

    /** 获取下标 index 处的字符 */
    char charAt(int index);

    /** 在 offset 位置前插入子串 t，返回新串 */
    IString insert(IString t, int offset);

    /** 删除 [begin, end) 范围内的字符，返回新串 */
    IString delete(int begin, int end);
}
💡 接口说明：顺序串（SeqString）和链串（LinkString）都实现此接口，因此文本编辑器的主逻辑使用 IString 类型引用，切换实现类时主程序无需改动。

实现顺序串类 SeqString 两版本不同

用字符数组存储，实现 IString 接口的 8 个操作

▼
📋 实训目标：用字符数组实现顺序串。顺序串将字符连续存储在 char[] 中，支持 O(1) 随机访问，适合频繁读取的场景。

顺序串用 char[] 存储字符，逻辑结构：

索引: 0 1 2 3 4 5 6 字符: S T U D E N T
src/string/SeqString.java 
/**
 * 顺序串实现类
 * 使用字符数组（char[]）存储，实现 IString 接口
 */
public class SeqString implements IString {

    public char[] data; // 字符数组，存储串的字符元素

    /** 用字符数组初始化 */
    public SeqString(char[] data) {
        this.data = new char[data.length];
        for (int i = 0; i < data.length; i++) {
            this.data[i] = data[i];
        }
    }

    /** 预分配指定长度的空串 */
    public SeqString(int length) {
        data = new char[length];
    }

    /** 返回串长度：统计非空字符数，O(n) */
    public int length() {
        int len = 0;
        for (int i = 0; i < data.length; i++) {
            if (data[i] != '\u0000') len++;
        }
        return len;
    }

    /** 联接：将串 t 接到当前串末尾，返回新串 */
    public IString concat(IString t) {
        int slen = this.length(), tlen = t.length();
        char[] buf = new char[slen + tlen];
        for (int i = 0; i < slen; i++) buf[i] = data[i];
        for (int j = 0; j < tlen; j++) buf[slen + j] = ((SeqString) t).data[j];
        return new SeqString(buf);
    }

    /** 比较：按字典序逐字符比较，返回 Unicode 差值 */
    public int compareTo(IString t) {
        char[] v1 = data, v2 = ((SeqString) t).data;
        int lim = Math.min(v1.length, v2.length);
        for (int i = 0; i < lim; i++) {
            if (v1[i] != v2[i]) return v1[i] - v2[i];
        }
        return v1.length - v2.length;
    }

    /** 取子串 [begin, end) */
    public IString substring(int begin, int end) {
        int subLen = end - begin;
        char[] buf = new char[subLen];
        for (int i = 0; i < subLen; i++) buf[i] = data[begin + i];
        return new SeqString(buf);
    }

    /** 定位（BF算法）：从 begin 开始查找子串 t */
    public int indexOf(IString t, int begin) {
        if (t == null || t.length() == 0 || this.length() < t.length()) return -1;
        int len = this.length(), tlen = t.length();
        SeqString t1 = (SeqString) t;
        int i = begin, j = 0;
        while (i < len && j < tlen) {
            if (data[i] == t1.data[j]) { i++; j++; }
            else { i = i - j + 1; j = 0; } // i 回退，j 重置
        }
        return (j >= tlen) ? i - tlen : -1;
    }

    /** 取值：O(1) 直接下标访问 */
    public char charAt(int index) { return data[index]; }

    /** 插入：在 offset 位置前插入子串 t */
    public IString insert(IString t, int offset) {
        int slen = this.length(), tlen = t.length();
        char[] buf = new char[slen + tlen];
        for (int i = 0; i < offset; i++) buf[i] = data[i];           // 第1段
        for (int i = 0; i < tlen; i++) buf[offset + i] = ((SeqString) t).data[i]; // 第2段
        for (int i = offset; i < slen; i++) buf[i + tlen] = data[i]; // 第3段
        return new SeqString(buf);
    }

    /** 删除：删除 [begin, end) 范围的字符 */
    public IString delete(int begin, int end) {
        if (end > this.length()) end = this.length();
        int len = end - begin;
        char[] buf = new char[this.length() - len];
        for (int i = 0; i < begin; i++) buf[i] = data[i];
        for (int i = end; i < this.length(); i++) buf[i - len] = data[i];
        return new SeqString(buf);
    }
}
💡 顺序串关键点： 插入/删除需创建新数组，时间复杂度 O(n)； charAt() 直接数组下标访问，O(1)； indexOf 使用 BF（暴力匹配）算法，最坏 O(n×m)。
📋 实训目标：用单链表实现链串。链串每个节点存一个字符，插入/删除只需调整指针，不需要移动大量字符，适合频繁修改的场景。

链串用 单链表 存储字符，逻辑结构（"abd"）：

start → [a|→] → [b|→] → [d|null]
src/string/LinkString.java 
/**
 * 链串实现类
 * 使用单链表存储，每个节点存一个字符，实现 IString 接口
 */
public class LinkString implements IString {

    /** 内部节点类 */
    private class Node {
        char data;
        Node next;
        public Node(char data, Node next) { this.data = data; this.next = next; }
    }

    private Node start; // 链串头指针

    /** 初始化空串 */
    public LinkString() { start = null; }

    /** 用字符数组初始化 */
    public LinkString(char[] data) {
        if (data == null || data.length == 0 || data[0] == '\u0000') { start = null; return; }
        Node node = new Node(data[0], null);
        start = node;
        Node p = start;
        for (int i = 1; i < data.length && data[i] != '\u0000'; i++) {
            node = new Node(data[i], null);
            p.next = node; p = node;
        }
    }

    /** 返回串长度：遍历链表统计节点数，O(n) */
    public int length() {
        int size = 0;
        Node p = start;
        while (p != null) { size++; p = p.next; }
        return size;
    }

    /** 联接：将 t 的链表接在当前串链表末尾 */
    public IString concat(IString t) {
        if (start == null) return t;
        Node p = start;
        while (p.next != null) p = p.next;
        p.next = ((LinkString) t).start; // 末尾接上 t 的头节点
        return this;
    }

    /** 比较：双指针并行遍历 */
    public int compareTo(IString t) {
        int slen = this.length(), tlen = t.length();
        Node sp = start, tp = ((LinkString) t).start;
        for (int i = 0; i < Math.min(slen, tlen); i++) {
            if (sp.data != tp.data) return sp.data - tp.data;
            sp = sp.next; tp = tp.next;
        }
        return slen - tlen;
    }

    /** 取子串 [begin, end) */
    public IString substring(int begin, int end) {
        int len = end - begin;
        char[] buf = new char[len];
        Node p = start;
        for (int i = 0; i < begin; i++) p = p.next;
        for (int i = 0; i < len; i++) { buf[i] = p.data; p = p.next; }
        return new LinkString(buf);
    }

    /** 定位（BF算法）：sp0 记录每次匹配起始位置 */
    public int indexOf(IString t, int begin) {
        if (t == null || this.length() < t.length()) return -1;
        Node sp = start;
        for (int i = 0; i < begin; i++) sp = sp.next;
        Node sp0 = sp;
        Node tp = ((LinkString) t).start;
        while (sp != null && tp != null) {
            if (sp.data == tp.data) { sp = sp.next; tp = tp.next; }
            else {
                if (sp0.next != null) { sp0 = sp0.next; sp = sp0; tp = ((LinkString) t).start; }
                else break;
            }
        }
        if (tp == null) { // 匹配成功，计算位置
            Node p = start; int pos = 0;
            while (p != sp0) { p = p.next; pos++; }
            return pos;
        }
        return -1;
    }

    /** 取值：需遍历到指定位置，O(n) */
    public char charAt(int index) {
        Node p = start;
        for (int i = 0; i < index; i++) p = p.next;
        return p.data;
    }

    /** 插入：找到 offset-1 节点，将 t 的链表插入 */
    public IString insert(IString t, int offset) {
        Node sp = start;
        for (int i = 0; i < offset - 1; i++) sp = sp.next;
        Node tp = sp;
        sp = sp.next;                       // sp 指向 offset 位置节点
        tp.next = ((LinkString) t).start; // tp.next 指向 t 头节点
        while (tp.next != null) tp = tp.next; // 遍历到 t 末尾
        tp.next = sp;                        // 连接原串后半段
        return this;
    }

    /** 删除：找到 begin-1 节点 p1 和 end 节点 p2，让 p1.next = p2 */
    public IString delete(int begin, int end) {
        if (end > this.length()) end = this.length();
        Node p1 = start;
        for (int i = 0; i < begin - 1; i++) p1 = p1.next;
        Node p2 = p1;
        for (int i = 0; i <= end - begin; i++) p2 = p2.next;
        p1.next = p2; // 跳过被删除的节点
        return this;
    }
}
💡 链串关键点： insert/delete 只需调整指针，不移动字符（已知节点位置时 O(1)）； charAt() 需从头遍历，O(n)； indexOf 用 sp0 记录每次匹配起始节点，回溯时重置为 sp0.next。

顺序串 vs 链串：操作对比 两版本对比

理解两种存储结构在各操作上的时间/空间差异

▼
📋 实训目标：通过代码对比，深刻理解同一接口在不同存储结构下的实现差异，以及对性能的影响。
操作 顺序串（SeqString） 链串（LinkString） 时间复杂度差异
length() 遍历数组统计非空字符 遍历链表统计节点数 均 O(n)
charAt(i) data[i] 直接访问 从 start 遍历 i 步 顺序 O(1)，链式 O(n) ⚡
insert(t, offset) 创建新数组，复制3段字符 调整 2 个指针即可 顺序 O(n)，链式 O(n) 找位置
delete(begin, end) 创建新数组，跳过被删区间 修改 p1.next = p2 顺序 O(n)，链式 O(1) 修改指针
indexOf(t, begin) 数组下标直接比较，BF算法 指针跳转比较，BF算法 均 O(n×m)，顺序串常数更小
concat(t) 创建新数组，复制两段 末节点.next = t.start 顺序 O(n)，链式 O(n) 找末尾
存储空间 仅字符，紧凑 每字符额外一个指针引用 链串约多 50% 开销
⚠️ 注意：链串的 concat() 和 insert() 会直接修改原串（破坏性操作），而顺序串返回新串。在文本编辑器中使用链串时，需注意这一差异。

📋 实训目标：实现完整的文本编辑器主程序，掌握用串操作完成实际文本处理任务的方法。 顺序串版本：所有 IString 实例用 new SeqString(...) 创建。
🔗 链串版本与顺序串版本的区别：仅将第 1 处构造语句 s = new SeqString(buf) 改为 s = new LinkString(buf)，以及各处 new SeqString(tc) 改为 new LinkString(tc)。其余所有 IString 接口调用完全不变。
src/TextEditor1.java（顺序串版） 
import java.io.*;
import java.util.Scanner;

/**
 * 文本编辑器主类（顺序串版本）
 * 功能：读取文件 / 统计字数 / 显示内容 / 统计词语
 *       追加文本 / 插入文本 / 替换文本 / 保存文本
 */
public class TextEditor1 {
    public static void main(String[] args) throws IOException {
        Scanner sc = new Scanner(System.in);
        
        // 显示功能菜单
        System.out.println("---------- 功能选择菜单----------");
        System.out.println("**********1. 读取文件**********");
        System.out.println("**********2. 统计字数**********");
        System.out.println("**********3. 显示内容**********");
        System.out.println("**********4. 统计词语**********");
        System.out.println("**********5. 追加文本**********");
        System.out.println("**********6. 插入文本**********");
        System.out.println("**********7. 替换文本**********");
        System.out.println("**********8. 保存文本**********");
        System.out.println("**********0. 退出程序**********");
        System.out.println("------------------------------");

        char ch;
        char[] buf = new char[150];
        IString s = null, st = null;  // s=原始文本, st=编辑后文本

        do {
            System.out.print("请输入您的选择：");
            ch = sc.next().charAt(0);

            switch (ch) {

                case '1': // ① 读取文件
                    FileReader fr = new FileReader("china.txt");
                    BufferedReader br = new BufferedReader(fr);
                    br.read(buf);
                    // ★ 顺序串版本：用字符数组构造 SeqString
                    s = new SeqString(buf);
                    System.out.println("读取文本文件成功！");
                    br.close();
                    break;

                case '2': // ② 统计字数
                    if (s != null)
                        System.out.println("文本中的字符数为：" + s.length());
                    break;

                case '3': // ③ 显示内容（每 80 字符换行）
                    if (s == null) break;
                    for (int i = 0; i < s.length(); i++) {
                        System.out.print(s.charAt(i));
                        if (i != 0 && i % 80 == 0 || i == s.length() - 1)
                            System.out.println();
                    }
                    break;

                case '4': // ④ 统计词语"祖国"出现次数
                    if (s == null) break;
                    char[] tc1 = {'祖', '国'};
                    // ★ 顺序串版本
                    IString t1 = new SeqString(tc1);
                    int count = 0, idx = 0;
                    while (idx < s.length()) {
                        int pos = s.indexOf(t1, idx);
                        if (pos != -1) { count++; idx = pos + tc1.length; }
                        else break;
                    }
                    System.out.println("词语在文本中出现次数：" + count);
                    break;

                case '5': // ⑤ 追加文本（在末尾追加"五星红旗迎风飘扬，胜利歌声多么响亮！"）
                    if (s == null) break;
                    char[] tc2 = {'五','星','红','旗','迎','风','飘','扬',
                                   '，','胜','利','歌','声','多','么','响','亮','！'};
                    // ★ 顺序串版本
                    IString t2 = new SeqString(tc2);
                    st = s.concat(t2);
                    System.out.println("追加完成，当前文本：");
                    printString(st);
                    break;

                case '6': // ⑥ 插入文本（在"五星红旗"前插入"\n"）
                    if (st == null) { System.out.println("请先执行操作5追加文本！"); break; }
                    char[] tc3 = {'五','星','红','旗'};
                    // ★ 顺序串版本
                    IString t3 = new SeqString(tc3);
                    int pos1 = st.indexOf(t3, 0);
                    char[] tc4 = {'\n'};
                    // ★ 顺序串版本
                    IString t4 = new SeqString(tc4);
                    st = st.insert(t4, pos1);
                    System.out.println("插入完成，当前文本：");
                    printString(st);
                    break;

                case '7': // ⑦ 替换文本（"胜利歌声" → "爱国歌声"）
                    if (st == null) { System.out.println("请先执行操作5和6！"); break; }
                    char[] tc5 = {'胜','利','歌','声'};
                    char[] tc6 = {'爱','国','歌','声'};
                    // ★ 顺序串版本
                    IString t5 = new SeqString(tc5);
                    IString t6 = new SeqString(tc6);
                    int pos2 = st.indexOf(t5, 0);
                    if (pos2 != -1) {
                        st = st.delete(pos2, pos2 + tc5.length);
                        st = st.insert(t6, pos2);
                        System.out.println("替换完成，当前文本：");
                        printString(st);
                    } else {
                        System.out.println("未找到目标字符串！");
                    }
                    break;

                case '8': // ⑧ 保存文本到 china1.txt
                    if (st == null) { System.out.println("没有可保存的内容！"); break; }
                    char[] stc = new char[st.length()];
                    for (int j = 0; j < st.length(); j++) stc[j] = st.charAt(j);
                    FileWriter fw = new FileWriter("china1.txt");
                    BufferedWriter bw = new BufferedWriter(fw);
                    bw.write(stc, 0, stc.length);
                    bw.close();
                    System.out.println("文本已保存到 china1.txt");
                    break;

                case '0':
                    System.out.println("程序退出。");
                    break;

                default:
                    System.out.println("无效选项，请重新输入。");
            }
        } while (ch != '0');

        sc.close();
    }

    /** 辅助方法：输出串内容（每80字符换行） */
    private static void printString(IString s) {
        for (int i = 0; i < s.length(); i++) {
            System.out.print(s.charAt(i));
            if (i != 0 && i % 80 == 0 || i == s.length() - 1)
                System.out.println();
        }
    }
}

思考题：串的模式匹配 两版本相同

用 KMP 算法优化文本编辑器的查找功能

▼
📋 题目：文本编辑器中的 indexOf 操作使用了 BF（暴力匹配）算法，最坏时间复杂度为 O(n×m)。请使用 KMP 算法重写 indexOf，将时间复杂度降到 O(n+m)。

KMP 算法核心： 利用模式串的 next 数组（最长前缀后缀数组），失配时不回退主串指针，只移动模式串指针。

KMP 模式匹配（参考实现） 
/**
 * KMP 模式匹配（Java 实现）
 * 时间复杂度 O(n+m)，主串指针不回退
 */
public static int kmpSearch(String text, String pattern) {
    int n = text.length(), m = pattern.length();
    if (m == 0) return 0;

    // ① 构建 next 数组（最长真前缀 = 最长真后缀 的长度）
    int[] next = buildNext(pattern);

    int j = 0; // 模式串指针
    for (int i = 0; i < n; i++) {       // i 遍历主串，永不回退
        while (j > 0 && text.charAt(i) != pattern.charAt(j)) {
            j = next[j - 1];           // 利用 next 回退模式串指针
        }
        if (text.charAt(i) == pattern.charAt(j)) j++;
        if (j == m) return i - m + 1;  // 匹配成功
    }
    return -1;
}

/** 构建 next 数组 */
private static int[] buildNext(String pattern) {
    int m = pattern.length();
    int[] next = new int[m];
    next[0] = 0;
    int k = 0;
    for (int i = 1; i < m; i++) {
        while (k > 0 && pattern.charAt(i) != pattern.charAt(k)) k = next[k - 1];
        if (pattern.charAt(i) == pattern.charAt(k)) k++;
        next[i] = k;
    }
    return next;
}

/*
 * 示例（模式串 "ABABCABAB" 的 next 数组）：
 *   j:    0  1  2  3  4  5  6  7  8
 *   t:    A  B  A  B  C  A  B  A  B
 *   next: 0  0  1  2  0  1  2  3  4
 *
 * 与 BF 对比：
 *   BF：   O(n×m) 最坏，主串指针会回退
 *   KMP：  O(n+m)，主串指针只前进，模式串利用 next 回退
 */

练习：将 TextEditor1 的 case '4'（统计词语）中的 indexOf 调用替换为 kmpSearch，对比两种算法在大文件（如 10 万字文章）中的运行时间差异。
