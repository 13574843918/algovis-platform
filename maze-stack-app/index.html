🔷 什么是栈（Stack）？

栈是一种后进先出（LIFO - Last In First Out）的线性数据结构。想象一下叠盘子：最后放上去的盘子，会最先被拿走。

核心操作：

• push - 入栈，放到顶部

• pop - 出栈，从顶部取出

• peek - 查看栈顶元素

算法原理

🗂️ 栈 LIFO 原理

栈（Stack）是后进先出（LIFO）的线性结构，迷宫探路的经典应用场景。

push入栈：将当前位置压入栈
pop出栈：回溯时弹出栈顶
peek查顶：查看当前探路位置
isEmpty空栈：表示无路可走
size大小：栈中元素个数

🔍 探路算法步骤

1

将起点压入栈，标记已访问

2

查看栈顶，尝试四个方向

3

有未访问通路则压栈前进

4

四方均不可走则出栈回溯

5

到终点或栈为空则终止

💻 核心伪代码

function solveMaze(maze, start, end): stack.push(start) visited[start] = true while !stack.isEmpty(): cur = stack.peek() if cur == end: return stack next = findNext(cur) if next: stack.push(next) else: stack.pop() // 回溯 return null

⬆️ push —— 入栈

将一个元素压入栈顶。这是栈最基本的操作，所有数据都从栈顶写入。

function push(stack, value): stack.data[stack.top + 1] = value stack.top = stack.top + 1 // 顺序栈实现 // JavaScript 数组实现 stack.push(value) // O(1)
时间复杂度：O(1) — 只需更新栈顶指针，不移动其他元素。
注意：顺序栈需检查是否上溢（栈满），链式栈无此限制。

⬇️ pop —— 出栈

移除并返回栈顶元素。只能从栈顶删除，不能从中间删除。

function pop(stack): if stack.isEmpty(): throw "栈下溢（Stack Underflow）" value = stack.data[stack.top] stack.top = stack.top - 1 return value // JavaScript 数组实现 const val = stack.pop() // O(1)
时间复杂度：O(1)
异常情况：对空栈执行 pop 会产生栈下溢（Underflow）错误，必须先判空！

👁️ peek / top —— 查看栈顶

返回栈顶元素的值，但不删除它。用于查看当前"最新"的元素。

function peek(stack): if stack.isEmpty(): throw "栈为空，无法查顶" return stack.data[stack.top] // top 指针不改变 // JavaScript 数组实现 stack[stack.length - 1] // O(1)
时间复杂度：O(1)
peek 与 pop 的区别：peek 只读，pop 读并删除。迷宫探路中频繁用 peek 获取当前位置。

❓ isEmpty —— 判断是否为空

检查栈中是否没有元素，是循环终止的关键判断条件。

function isEmpty(stack): return stack.top == -1 // 顺序栈：top=-1 表示空 // JavaScript 数组实现 stack.length === 0 // O(1) // 迷宫中的典型用法 while (!stack.isEmpty()) { // 继续探路... }
时间复杂度：O(1)
迷宫算法中，isEmpty 返回 true 意味着所有路径都已探索完毕，迷宫无解。

📏 size —— 获取栈的大小

返回栈中当前存储的元素个数。在迷宫问题中，size 等于当前路径长度。

function size(stack): return stack.top + 1 // 顺序栈：top从-1开始 // JavaScript 数组实现 stack.length // O(1) // 迷宫中：size = 已走路径步数 pathLen = stack.size()
时间复杂度：O(1)
找到终点时，stack.size() 就是最短路径长度。在迷宫中 size 也间接反映了探索深度。

💻 完整栈实现代码

class Stack { constructor() { this.data = []; // 存储数组 } // 入栈 O(1) push(value) { this.data.push(value); } // 出栈 O(1) pop() { if (this.isEmpty()) throw new Error("Stack Underflow"); return this.data.pop(); } // 查看栈顶 O(1) peek() { if (this.isEmpty()) throw new Error("Stack is empty"); return this.data[this.data.length - 1]; } // 判断是否为空 O(1) isEmpty() { return this.data.length === 0; } // 获取大小 O(1) size() { return this.data.length; } // 清空 O(1) clear() { this.data = []; } } // 使用示例 const s = new Stack(); s.push(1); s.push(2); s.push(3); console.log(s.peek()); // 3 console.log(s.pop()); // 3 console.log(s.size()); // 2 console.log(s.isEmpty()); // false
class Stack: def __init__(self): self.data = [] # 使用列表实现 # 入栈 O(1) def push(self, value): self.data.append(value) # 出栈 O(1) def pop(self): if self.is_empty(): raise IndexError("Stack Underflow") return self.data.pop() # 查看栈顶 O(1) def peek(self): if self.is_empty(): raise IndexError("Stack is empty") return self.data[-1] # 判断是否为空 O(1) def is_empty(self): return len(self.data) == 0 # 获取大小 O(1) def size(self): return len(self.data) # 使用示例 s = Stack() s.push(1); s.push(2); s.push(3) print(s.peek()) # 3 print(s.pop()) # 3 print(s.size()) # 2 print(s.is_empty()) # False
// C 语言 顺序栈实现 #define MAXSIZE 100 typedef struct { int data[MAXSIZE]; int top; // 栈顶指针，初始为 -1 } Stack; // 初始化 void initStack(Stack *s) { s->top = -1; } // 判断空 O(1) int isEmpty(Stack *s) { return s->top == -1; } // 判断满 O(1) int isFull(Stack *s) { return s->top == MAXSIZE - 1; } // 入栈 O(1) int push(Stack *s, int val) { if (isFull(s)) return 0; // 上溢 s->data[++s->top] = val; return 1; } // 出栈 O(1) int pop(Stack *s, int *val) { if (isEmpty(s)) return 0; // 下溢 *val = s->data[s->top--]; return 1; } // 查顶 O(1) int peek(Stack *s, int *val) { if (isEmpty(s)) return 0; *val = s->data[s->top]; return 1; } // 获取大小 O(1) int size(Stack *s) { return s->top + 1; }
// C 语言 链式栈实现 typedef struct Node { int data; struct Node *next; } Node; typedef struct { Node *top; // 栈顶指针 int size; } LinkedStack; // 初始化 void initStack(LinkedStack *s) { s->top = NULL; s->size = 0; } // 判断空 O(1) int isEmpty(LinkedStack *s) { return s->top == NULL; } // 入栈 O(1) — 无上溢风险 void push(LinkedStack *s, int val) { Node *node = (Node*)malloc(sizeof(Node)); node->data = val; node->next = s->top; s->top = node; s->size++; } // 出栈 O(1) int pop(LinkedStack *s) { if (isEmpty(s)) return -1; Node *tmp = s->top; int val = tmp->data; s->top = tmp->next; free(tmp); s->size--; return val; }

用栈实现迷宫探路

深入理解栈（Stack）数据结构在经典迷宫问题中的应用，掌握深度优先搜索（DFS）的核心思想

什么是栈？

栈是一种线性数据结构，遵循后进先出（LIFO）原则，像叠盘子——最后放的最先取。

(2,3) ← 栈顶 TOP
(1,3)
(1,2)
(0,0) ← 栈底 BOTTOM

核心操作：push、pop、peek，时间复杂度均为 O(1)。

为什么用栈解迷宫？

迷宫探路本质是深度优先搜索（DFS），栈天然支持"走到死路时回溯"：

  • 走一步 → push 新位置
  • 死路 → pop 回溯
  • 找到终点时栈内即完整路径
  • 栈为空 → 迷宫无解

✓ 栈解迷宫 = 手动实现 DFS 的系统调用栈

完整代码实现

const DIRS=[[-1,0],[1,0],[0,-1],[0,1]]; function solveMaze(maze, start, end) { const stack=[start]; const visited=new Set([start.toString()]); while(stack.length>0) { const [r,c]=stack[stack.length-1]; if(r===end[0]&&c===end[1]) return stack; let moved=false; for(const [dr,dc] of DIRS){ const [nr,nc]=[r+dr,c+dc]; if(maze[nr][nc]===0&&!visited.has(`${nr},${nc}`)){ stack.push([nr,nc]); visited.add(`${nr},${nc}`); moved=true; break; } } if(!moved) stack.pop(); } return null; }
