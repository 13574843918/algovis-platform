实现栈接口 IStack
▼
📋 实训目标：定义栈的抽象数据类型，为后续实现做好准备。

栈是一种后进先出（LIFO）的数据结构，只允许在一端进行插入和删除操作。

src/stack/IStack.java 
/**
 * 栈接口定义
 * 定义栈的基本操作方法
 * @param <E> 泛型参数，表示栈中元素的类型
 */
public interface IStack<E> {
    
    /**
     * 初始化栈
     * 将栈顶指针设置为初始状态
     */
    void init();
    
    /**
     * 元素入栈（压栈）
     * 将元素添加到栈顶
     * @param element 要入栈的元素
     * @return 入栈成功返回元素本身，失败返回null
     */
    E push(E element);
    
    /**
     * 元素出栈（弹栈）
     * 移除并返回栈顶元素
     * @return 出栈的元素，栈空时返回null
     */
    E pop();
    
    /**
     * 查看栈顶元素（不删除）
     * @return 栈顶元素，栈空时返回null
     */
    E peek();
    
    /**
     * 获取栈中元素个数
     * @return 元素个数
     */
    int size();
    
    /**
     * 判断栈是否为空
     * @return 栈空返回true，否则返回false
     */
    boolean empty();
}
💡 接口说明：接口只定义方法签名，实现类负责具体逻辑。使用泛型 <E> 可以存储任意类型的数据。

实现顺序栈 SeqStack
▼
📋 实训目标：使用数组实现栈的顺序存储结构。

顺序栈使用数组存储元素，通过栈顶指针 top追踪栈的状态。

src/stack/SeqStack.java 
/**
 * 顺序栈实现类
 * 使用数组存储元素，实现IStack接口
 * @param <E> 泛型参数
 */
import java.lang.reflect.Array;

public class SeqStack<E> implements IStack<E> {
    
    // 存储数据的数组
    private E[] data;
    
    // 栈顶指针，-1表示空栈
    private int top;
    
    // 栈的最大容量
    private int maxSize;
    
    /**
     * 构造函数
     * @param type 元素类型的Class对象（用于泛型数组创建）
     * @param size 栈的容量
     */
    public SeqStack(Class<E> type, int size) {
        // 使用反射创建泛型数组
        data = (E[]) Array.newInstance(type, size);
        maxSize = size;
        top = -1; // 栈空时top为-1
    }
    
    /**
     * 初始化栈
     */
    @Override
    public void init() {
        top = -1;
    }
    
    /**
     * 入栈操作
     * @param element 要入栈的元素
     * @return 入栈成功返回元素，失败返回null
     */
    @Override
    public E push(E element) {
        // 检查栈是否已满
        if (top == maxSize - 1) {
            return null; // 栈满，入栈失败
        }
        data[++top] = element; // 先移动指针，再存入数据
        return element;
    }
    
    /**
     * 出栈操作
     * @return 出栈的元素，栈空时返回null
     */
    @Override
    public E pop() {
        // 检查栈是否为空
        if (top == -1) {
            return null; // 栈空，出栈失败
        }
        return data[top--]; // 先返回数据，再移动指针
    }
    
    /**
     * 查看栈顶元素（不删除）
     * @return 栈顶元素，栈空时返回null
     */
    @Override
    public E peek() {
        if (top == -1) {
            return null;
        }
        return data[top];
    }
    
    /**
     * 获取栈中元素个数
     * @return 元素个数
     */
    @Override
    public int size() {
        return top + 1;
    }
    
    /**
     * 判断栈是否为空
     * @return 栈空返回true，否则返回false
     */
    @Override
    public boolean empty() {
        return top == -1;
    }
    
    /**
     * 获取栈的当前状态（用于调试）
     * @return 栈的状态信息
     */
    public String getStatus() {
        return String.format("栈状态：大小=%d，容量=%d，栈顶位置=%d", 
                           size(), maxSize, top);
    }
}
⚠️ 关键点：数组索引从0开始，所以空栈时 top = -1，入栈时先 ++top，出栈时先返回数据再 top--。

实现路径点类 Point
▼
📋 实训目标：创建坐标点类，用于记录迷宫中的位置和搜索方向。

每个 Point 对象记录一个迷宫位置，以及从这个位置已经尝试过的方向。

src/maze/Point.java 
/**
 * 迷宫路径点类
 * 用于记录迷宫中的位置和搜索方向
 */
public class Point {
    
    // 行坐标
    public int x;
    
    // 列坐标
    public int y;
    
    // 当前试探的方向（0:东, 1:南, 2:西, 3:北，-1:初始）
    // 方向示意：0=右，1=下，2=左，3=上
    public int d;
    
    /**
     * 构造函数
     * @param x 行坐标
     * @param y 列坐标
     * @param d 方向
     */
    public Point(int x, int y, int d) {
        this.x = x;
        this.y = y;
        this.d = d;
    }
    
    /**
     * 重写toString方法，便于输出调试
     * @return 点的字符串表示，如 (1,2,3)
     */
    @Override
    public String toString() {
        return String.format("(%d,%d,%d)", x, y, d);
    }
}
💡 方向编码：
• d = 0 表示已尝试向东（右）
• d = 1 表示已尝试向南（下）
• d = 2 表示已尝试向西（左）
• d = 3 表示已尝试向北（上）
• d = -1 表示刚到达此位置，尚未尝试任何方向

实现迷宫类 Maze
▼
📋 实训目标：创建迷宫类，负责迷宫地图的初始化、路径标记和数据访问。

迷宫使用二维数组表示：0=墙壁，1=通路，2=已走过，3=最终路径。

src/maze/Maze.java 
/**
 * 迷宫类
 * 负责迷宫的初始化和路径搜索
 */
public class Maze {
    
    // 迷宫地图二维数组
    private int[][] maze;
    
    // 迷宫行数（包含外墙）
    private int rows;
    
    // 迷宫列数（包含外墙）
    private int cols;
    
    // 路径栈，用于记录走过的路径
    private IStack<Point> path;
    
    /**
     * 构造函数
     * @param map 原始迷宫地图（不含外墙）
     */
    public Maze(int[][] map) {
        // 计算迷宫大小：外围加一圈墙壁
        rows = map.length + 2;      // 原高度 + 上下外墙
        cols = map[0].length + 2;  // 原宽度 + 左右外墙
        
        // 初始化迷宫数组
        maze = new int[rows][cols];
        
        // 复制原始地图到中间位置，边缘自动为0（墙壁）
        for (int i = 1; i < rows - 1; i++) {
            for (int j = 1; j < cols - 1; j++) {
                maze[i][j] = map[i - 1][j - 1];
            }
        }
        
        // 初始化路径栈
        path = new SeqStack<>(Point.class, rows * cols);
    }
    
    /**
     * 打印迷宫地图
     */
    public void printMaze() {
        System.out.println("迷宫地图：");
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                System.out.printf("%2d ", maze[i][j]);
            }
            System.out.println();
        }
    }
    
    /**
     * 获取路径栈
     * @return 路径栈
     */
    public IStack<Point> getPath() {
        return path;
    }
    
    /**
     * 获取迷宫行数
     */
    public int getRows() {
        return rows;
    }
    
    /**
     * 获取迷宫列数
     */
    public int getCols() {
        return cols;
    }
    
    /**
     * 获取指定位置的值
     * @param x 行坐标
     * @param y 列坐标
     * @return 位置值（0=墙，1=路）
     */
    public int getValue(int x, int y) {
        return maze[x][y];
    }
    
    /**
     * 设置指定位置的值
     * @param x 行坐标
     * @param y 列坐标
     * @param value 要设置的值
     */
    public void setValue(int x, int y, int value) {
        maze[x][y] = value;
    }
}
💡 外墙设计：在原始迷宫外围加一圈墙壁（值为0），这样搜索时不需要额外判断边界是否越界。

实现迷宫求解器 MazeSolver
▼
📋 实训目标：实现基于栈的迷宫路径搜索算法，理解回溯算法的核心思想。

这是核心算法部分！使用回溯法搜索路径：沿着一条路走到底，如果死路就回溯到上一个岔口尝试其他方向。

src/maze/MazeSolver.java 
/**
 * 迷宫求解器
 * 实现基于栈的迷宫路径搜索算法
 */
import java.util.ArrayList;
import java.util.List;

public class MazeSolver {
    
    /**
     * 求解迷宫路径（核心算法）
     * 使用回溯法：沿一个方向走到底，不通则回退尝试其他方向
     * @param maze 迷宫对象
     * @return 是否找到路径
     */
    public static boolean solveMaze(Maze maze) {
        IStack<Point> path = maze.getPath();
        int rows = maze.getRows();
        int cols = maze.getCols();
        
        // 步骤1：从入口(1,1)开始搜索
        Point startPoint = new Point(1, 1, -1);
        path.push(startPoint);
        maze.setValue(1, 1, 2); // 标记起点为已走过
        System.out.println("开始搜索路径...");
        
        // 步骤2：主循环 - 栈不为空就一直搜索
        while (!path.empty()) {
            
            // 步骤3：弹出栈顶元素（回溯的核心）
            Point currentPoint = path.pop();
            int x = currentPoint.x;
            int y = currentPoint.y;
            int d = currentPoint.d + 1; // 从上次失败的下一个方向开始
            
            System.out.printf("当前位置：(%d,%d)，上次方向：%d\n", x, y, currentPoint.d);
            
            // 步骤4：按顺序尝试四个方向
            boolean found = false; // 是否找到通路
            while (d < 4 && !found) {
                int nextX = x;
                int nextY = y;
                
                // 根据方向计算下一个位置
                switch (d) {
                    case 0: // 东（右）
                        nextY = y + 1;
                        break;
                    case 1: // 南（下）
                        nextX = x + 1;
                        break;
                    case 2: // 西（左）
                        nextY = y - 1;
                        break;
                    case 3: // 北（上）
                        nextX = x - 1;
                        break;
                }
                
                System.out.printf("尝试方向 %d -> (%d,%d)\n", d, nextX, nextY);
                
                // 步骤5：检查下一个位置是否可通行
                if (maze.getValue(nextX, nextY) == 1) {
                    
                    // 步骤6：可通行！将当前点按新方向重新入栈
                    path.push(new Point(x, y, d));
                    
                    // 步骤7：更新坐标，标记该点已走过
                    x = nextX;
                    y = nextY;
                    maze.setValue(x, y, 2);
                    
                    // 步骤8：将新位置入栈
                    path.push(new Point(x, y, -1));
                    System.out.printf("移动到新位置：( %d,%d)\n", x, y);
                    
                    // 步骤9：检查是否到达出口
                    if (x == rows - 2 && y == cols - 2) {
                        System.out.println("找到出口！");
                        return true;
                    }
                    
                    found = true; // 找到通路，跳出方向循环
                } else {
                    d++; // 此方向不通，尝试下一个方向
                }
            }
            
            // 步骤10：四个方向都不通，自动回溯（栈已弹出当前点）
            if (!found) {
                System.out.printf("位置(%d,%d)四个方向都不通，回溯\n", x, y);
            }
        }
        
        System.out.println("没有找到路径！");
        return false;
    }
    
    /**
     * 打印找到的路径
     * @param maze 迷宫对象
     */
    public static void printPath(Maze maze) {
        IStack<Point> path = maze.getPath();
        
        if (path.empty()) {
            System.out.println("没有找到路径！");
            return;
        }
        
        System.out.println("\n找到的路径（从入口到出口）：");
        
        // 栈是LIFO，需要倒序输出
        List<Point> pathList = new ArrayList<Point>();
        while (!path.empty()) {
            pathList.add(0, path.pop()); // 从列表开头插入，实现倒序
        }
        
        for (int i = 0; i < pathList.size(); i++) {
            Point p = pathList.get(i);
            System.out.printf("步骤%d: (%d,%d)\n", i+1, p.x, p.y);
        }
        
        // 重新将路径压回栈中（用于后续显示）
        for (int i = pathList.size() - 1; i >= 0; i--) {
            path.push(pathList.get(i));
        }
    }
    
    /**
     * 可视化显示迷宫路径
     * @param maze 迷宫对象
     */
    public static void visualizeMaze(Maze maze) {
        int rows = maze.getRows();
        int cols = maze.getCols();
        
        System.out.println("\n迷宫路径可视化：");
        System.out.println("■=墙壁  ·=通路  ■=最终路径");
        
        // 创建可视化数组
        int[][] visual = new int[rows][cols];
        
        // 复制原始迷宫状态
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                visual[i][j] = maze.getValue(i, j);
                if (visual[i][j] == 2) {
                    visual[i][j] = 1; // 已走过的路径恢复为普通通路
                }
            }
        }
        
        // 标记最终路径
        IStack<Point> path = maze.getPath();
        List<Point> pathList = new ArrayList<Point>();
        while (!path.empty()) {
            pathList.add(0, path.pop());
        }
        for (Point p : pathList) {
            visual[p.x][p.y] = 3; // 最终路径标记为3
        }
        // 重新压回路径
        for (int i = pathList.size() - 1; i >= 0; i--) {
            path.push(pathList.get(i));
        }
        
        // 打印可视化结果
        String RESET = "\u001B[0m";
        String RED = "\u001B[31m";
        String GREY = "\u001B[90m";
        
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                switch (visual[i][j]) {
                    case 0: // 墙壁
                        System.out.print(GREY + "■" + RESET + " ");
                        break;
                    case 1: // 普通通路
                        System.out.print("· ");
                        break;
                    case 3: // 最终路径
                        System.out.print(RED + "■" + RESET + " ");
                        break;
                    default:
                        System.out.print("· ");
                }
            }
            System.out.println();
        }
    }
}
⚠️ 回溯算法关键：
1. 当四个方向都不通时，当前点已经出栈（pop），自动回到上一个岔口
2. 每个点记录上次尝试的方向（d），下次从 d+1 开始尝试，避免重复
3. 出口是 (rows-2, cols-2)，因为我们加了外墙

思考题：下一个更大元素
▼
📋 题目：给定一个长度为 n 的整数数组，找出每个元素右边第一个比它大的数。如果没有比它大的数，则输出 -1。

要求：

时间复杂度为 O(n)（不能用双层循环）。
提示：用栈解决这个问题。

示例：

输入: [2, 1, 5, 6, 2, 3] 输出: [5, 5, -1, -1, 3, -1] 解释: - 2的右边第一个比它大的数是5 - 1的右边第一个比它大的数是5 - 5的右边没有比它大的数，所以是-1 - 6的右边没有比它大的数，所以是-1 - 2的右边第一个比它大的数是3 - 3的右边没有比它大的数，所以是-1
